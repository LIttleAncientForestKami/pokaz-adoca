== UMLet

Not a fan. First, for a long time (and some unknown reason) UMLet didn't work for me. :( 
When I finally made it work, I was underwhelmed. Lot of writing to get that UML, when compared to, say, PlantUML.

WARNING: SVG vs PNG perhaps?

=== Setup

. I installed the standalone app from https://www.umlet.com/
.. Download: https://www.umlet.com/changes.htm
.. Install by unzip, for instance to `/opt/Umlet`,
.. `chmod +x umlet.sh`
.. Add a `/usr/bin/umlet` pointing at `umlet.sh` and you're set.
. Troubleshooting
.. I corrected the incorrect UMLet home var so JAR is reached
.. Operate on local files, not on remote ones
